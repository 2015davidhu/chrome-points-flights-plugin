# Personal Fitness Tracker

A simple fitness tracking app built with React + TypeScript + Vite.

## Features

- Log workouts with type, duration, calories, and notes
- View workout history
- See summary stats (total workouts, minutes, calories)
- Data persisted in localStorage

## Getting Started

```bash
cd personal-fitness-tracker
npm install
npm run dev
```

## Splitting into its own repo

This project is self-contained. To move it to its own repository:

```bash
cp -r personal-fitness-tracker /path/to/new/repo
cd /path/to/new/repo
git init
git add .
git commit -m "Initial commit"
```
